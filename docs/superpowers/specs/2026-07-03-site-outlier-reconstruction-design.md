# 观我 Fate OS 出圈级重构方案

Date: 2026-07-03
Status: First implementation wave in progress

## Final Positioning

观我不应该再被表达成“工具很多的玄学网站”，而应该被表达成：

> 观我 Fate OS：把命运分析变成今天能执行的行动系统。

核心差异不是比竞品算得更多，而是把八字、紫微、星盘、塔罗、面相手相、真实问题和用户反馈合并到同一个上下文里，输出可执行、可复盘、可延展到关系与生活方式选择的行动闭环。

## Competitor Reading

参考的竞品方向：

- Co-Star: 强品牌语气、社交传播、每日推送。
- The Pattern: 深度人格模式、关系洞察、长期陪伴感。
- CHANI: 今日/本周/年度节奏、仪式感、会员内容。
- Sanctuary: 真人咨询、清晰付费转化、信任背书。
- 国内同类产品: 通常强在工具矩阵、星盘/塔罗/八字入口、AI 问答和泛社区，但容易变成同质化“测一测”集合。

观我的机会是把“工具集合”升级为“个人命运操作系统”。免费工具负责破冰，完整画像负责建档，今日行动负责留存，关系合参负责传播，藏宝阁处方负责商业闭环。

## Product Architecture

### 1. Free Exploration Layer

保留星际分析和性格测验，但放到工具层，不再作为主品牌承诺。

用途：

- 低门槛体验。
- 分享裂变。
- 收集第一层偏好和反馈。
- 引导用户进入完整画像。

入口：

- `/tools`
- `/divination`
- `/am16`
- `/almanac`

### 2. Core Profile Layer

完整命运行动图是核心付费产品。

它必须回答：

- 我当前处在什么模式？
- 我的机会窗口是什么？
- 我的主要盲点是什么？
- 今天和本周具体做什么？
- 哪些关系、事业、财富、健康、心智信号互相冲突？

### 3. Relationship Sync Layer

关系合参独立成产品，不藏在单主题分析里。

它承担：

- 恋爱、婚姻、复合、合作、家庭沟通。
- 更强的分享动机。
- 更容易让一个用户带来另一个用户。

### 4. Daily Command Layer

今日行动板是留存核心。

它不能只是黄历或每日趋势，而要变成：

- 今日主题。
- 今日风险。
- 今日最值得推进的一件事。
- 一条可复盘行动。

### 5. Treasure Prescription Layer

藏宝阁不是普通商城，而是报告后的处方电商。

它要承接：

- 报告弱点。
- 五行补强。
- 星象阶段。
- 当前成长任务。
- 用户当下问题。

表达边界：

- 不能承诺确定功效。
- 必须强调文化创意、生活方式建议和理性选择。

## Homepage Reconstruction

首屏必须在 5 秒内讲清楚：

- 观我是什么：AI 命运行动系统。
- 不是算命终点：是报告、今日行动、关系合参、藏宝阁处方的闭环。
- 免费工具还在，但不是主线。
- 用户第一步应该生成完整命运行动图。

第一波已经执行：

- 首页 Hero 改为 Fate OS 命运行动系统。
- 产品路径从 4 个扩成 6 个：免费探索、单主题快问、完整命运行动图、关系合参、今日行动板、藏宝阁处方。
- 首页分区标题统一为“信号 -> 行动 -> 处方”的闭环叙事。
- 站点 metadata、slogan、导航、页脚同步新定位。

## Next Implementation Waves

### P1: Reading Flow

重构 `/reading/new`：

- 先问用户真实问题。
- 再选择分析深度。
- 最后补出生信息和可选图片。
- 单主题快问必须输出主题切片，不能伪装成完整报告。

### P2: Dashboard Retention

重构登录后首页：

- 今日行动板置顶。
- 报告档案、历史复盘和下一步建议同屏。
- 让用户每天回来“执行和打卡”，不是只查看静态报告。

### P3: Treasure Hall Prescription Commerce

继续强化 `/shop`：

- 根据报告标签解释推荐理由。
- 非画像用户用需求路径浏览。
- 商品卡必须显示匹配理由、使用建议和风险边界。

### P4: Viral Artifact

生成可分享的结果卡：

- 今日行动卡。
- AM16 结果卡。
- 关系合参摘要卡。
- 创始席位/邀请卡。

### P5: Content And Localization QA

全站检查：

- 历史“AI 行为分析镜”遗留命名。
- 中英文术语一致性。
- SEO 页面标题。
- 法务边界文案。

## Decision On Star Analysis And Personality Quiz

保留。

但定位必须改变：

- 不是核心产品。
- 不是主导航的第一承诺。
- 是免费工具层和裂变入口。
- 结果页必须引导到完整命运行动图、今日行动板或关系合参。
